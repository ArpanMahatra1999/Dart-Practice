# Dart-Practice
Dart files describe different dart programming language features.

Tools used:
1. Online Dart IDE
2. Notepad
3. Command Prompt

Languages used:
1. Dart 2.9.1

Steps to Install:
1. Go to Command Prompt
2. Change directory to files location using 'cd FILE_LOCATION'
3. Execute 'dart FILE_NAME.dart' in command prompt.
